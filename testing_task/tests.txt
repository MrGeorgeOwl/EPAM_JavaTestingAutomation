1. Адаптивность сайта для пк
 - Открыть консоль разработчика
 - Открыть панель устройств
 - Менять разрешение экрана
 Assert: сайт должен выглядеть адекватно для большинства
 разрешений и информация должна быть удобочитаемая, компо-
 ненты сайта не налазят друг на друга

2. Адаптивный сайт для мобильной версии сайта
 - Открыть сайт на мобильном устройстве
 Assert: сайт должен сохранить функциональность версии для компьютера,
 сайт должен быть удобочитаемым, качество картинок не шакальным, компо-
 ненты сайта выстроены в колонку

3. Добавление товара в корзину
 - Выбрать интересующий товар 
 - Нажать на кнопку "Выбрать"
 - Выбрать комплектацию товара
 - Нажать на "Добавить в корзину"
 Assert: товар должен появиться в корзине с стоимостью, которая была получена
 после редактирования комплектации товара

4. Удаление товара из корзины
 - Проделать все шаги из предыдущего тестового случая
 - Навести курсор на "Корзина"
 - В отображенном списке кликнуть на иконку корзины интересующего товара
 Assert: товар должен быть удален из списка корзины

5. Редактирование заказа на этапе корзины
5.1 Редактирование не требуется
 - Проделать все шаги третьего тестового случая
 - Нажать на кнопку "Корзина"
 - Перейдя на новую страницу, проверить комплектацию заказа
 - Нажать "Оформить заказ"
 Assert: должно появится окно с указанием адреса доставки

5.2 Редактирование требуется
 - Проделать все шаги третьего тестового случая
 - Нажать на кнопку "Корзина"
 - Перейдя на новую страницу, проверить комплектацию заказа
 - Удалить ненужный товар
 - Нажать "Оформить заказ"
 Assert: цена должна поменяться в соответствии с удаленным товаром, отобразиться
 окно с указанием адреса

5.3 При редактировании происходит добавление предмета
 - Проделать все шаги третьего случая
 - Нажать на кнопку "Корзина"
 - Выбрать что-нибудь из соусов(Ранч топ, кста)
 - Нажать "Оформить заказ"
 Assert: цена должна поменяться в соответствии с выбранным соусом/соусами, отобразиться
 окно с указанием адреса

5.4 При редактировании происходит активация промокода 
 - Проделать все шаги третьего случая
 - Нажать на кнопку "Корзина"
 - Использовать промокод
 - Нажать "Оформить заказ"
 Assert: цена должна поменяться в соответствии с эффектом промокода, отобразиться
 окно с указанием адреса

6. Заполнение формы указания адрес не полностью
 - Проделать все шаги третьего случая
 - Проделать все шаги одного из пунктов 5.1-5.4
 - В появившейся форме не заполнить одно из полей: Улица, Дом, Квартира, Подъезд
 - Нажать "Потвердить адрес"
 Assert: Форма должна выдать ошибку и пометить поля, которые требуется заполнить

7. Заполнение формы указания адреса полностью
 - Проделать все шаги третьего случая
 - Проделать все шаги одного из пунктов 5.1-5.4
 - В появившейся форме заполнить все поля: Улица, Дом, Квартира, Подъезд
 - Нажать "Потвердить адрес"
 Assert: форма должна закрыться успешно и пользователь должен перейти на новую страницу,
 где сказано, что ему позвонят для потверждения заказа

8. Выбор точки для самовывоза
 - Проделать все шаги третьего случая
 - Проделать все шаги одного из пунктов 5.1-5.4
 - Нажать на "Самовывоз"
 - Выбрать одну из точек
 Assert: форма не должна позволять юзеру выбрать несколько пунктов за раз,
 после потверждения должен перейти переход на страницу, на которой сказано,
 что юзеру позвонят, чтобы потвердить заказ

9. Перелистывания главной страницы при нажатии на соответствующую категорию
 - Перейти на главную страницу
 - Нажать на интересующую категорию
 Assert: Страница должна перейти ниже на соответствующую категории, где указаны её товары

10. Форма для авторизации на сайте
10.1 Правильное заполнение
 - Перейти на главную страницу
 - Нажать на "Войти"
 - Ввод валидного телефона
 - Нажать "Выслать код"
 Assert: пользователь должен получить код, при введении которого, переходит на страницу
 редактирования профиля

10.2 Зполнение невалидным телефоном
 - Перейти на главную страницу
 - Нажать на "Войти"
 - Ввод невалидного телефона
 - Нажать "Выслать код"
 Assert: форма должна подсветить поле и ответить, что пользователь указал неправильный телефон

10.3 Заполнение формы неполностью:
 - Перейти на главную страницу
 - Нажать на "Войти"
 - Нажать "Выслать код"
 Assert: форма должна подсветить поле и ответить, что пользователь не ввел свой телефон
